# arrayadditionwithlooping.s
- Initializes a new array with length 1-9. 
- Creates another array with has the sum of all previous values from the previous array.
i.e  
p[0] = 9, p[1] = 2, p[2] = 3 then q[2] = p[0] + p[1] + p[2].
